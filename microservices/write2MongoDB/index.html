Welcome to one of our GFK Dataflow Control API's

General Info:

  • This API connects to one of GFK's servers and reads logs of the processes that run on a daily basis for all the countries.

    • Parameter Info:

  • process = The GFK Application that runs (for example: Porthos, Aramis, Datatrans, RC)
  • isoCode = The country's ISOCode (for example: RO=Romania, AT=Austria, etc.)
  • date = The date from which we want the protocols to be displayed YYYYmmdd format (for example: 20180922 )
  • filename = The exact name of the log file produced (for example: ScanPlus_dataro_20180920_062118.xml.log )

    • sendLogData/porthos , data

  • Details : This is an API does a post to the database with the process name and the JSON data it wants to write to the db.
  • Example : sendLogData/porthos , [data]
