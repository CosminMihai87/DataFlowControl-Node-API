Welcome to one of our GFK Dataflow Control API's

General Info:

  • This API connects to one of GFK's servers and reads logs of the processes that run on a daily basis for all the countries.

    • Parameter Info:

  • process = The GFK Application that runs (for example: Porthos, Aramis, Datatrans, RC)
  • isoCode = The country's ISOCode (for example: RO=Romania, AT=Austria, etc.)
  • date = The date from which we want the protocols to be displayed YYYYmmdd format (for example: 20180922 )
  • filename = The exact name of the log file produced (for example: ScanPlus_dataro_20180920_062118.xml.log )

    • /getLogFilesList/:process/:isoCode

  • Details : This is an API that returns a JSON array with all the process runs in the curent day for the specified application and country.
  • Example : getLogFilesList/Porthos/RO/

    • /getLogFilesList/:process/:isoCode/:date

  • Details : This is an API that returns a JSON array with all the process runs for the specified day,application and country.
  • Example : getLogFilesList/Porthos/RO/20180922

    • /getLogData/:process/:isoCode/:filename

  • Details : This is an API that returns a JSON with all the process details of a specific run extracted from a specific log file for a specific Country and Application
  • Example : getLogData/Porthos/RO/ScanPlus_RO_RC_20180927_062355.log
